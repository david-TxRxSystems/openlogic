Features for Professional Integrators

Modular Architecture

Create reusable logic blocks and structured modules that scale across rooms, subsystems, and hardware platforms.

Clean Signal Logic

Inspired by SIMPL+ — refined for modern applications. Digital, analog, and serial signals are first-class citizens.

Logic Symbol Library

Drop-in building blocks like AND, TOGGLE, LATCH, TIMERS, COUNTERS, and more — already scaffolded and ready to use.

Custom UI Designer

A browser-based layout engine is coming. Drag and drop buttons, sliders, labels — and bind them directly to logic signals.

System Integration

Built-in bridges to Home Assistant, Crestron Home, Lutron, Dynalite, and more. Extend or replace legacy control with ease.

Portable Runtime

Runs in Docker or Linux. Deploy to a NUC, VM, or Pi. Low overhead, maximum flexibility.

Built in Public

Transparent roadmap, open GitHub repo, and a growing community of professionals contributing real-world insight.
