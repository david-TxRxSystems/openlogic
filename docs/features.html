Features for Professional Integrators

Features for Real Integrators

Modular Architecture

Each room, device, or subsystem is its own clean module — wired together using a declarative logic language built for integrators.

Visual Logic Designer

Drag-and-drop programming interface that stays in sync with the DSL code behind it. Familiar wiring, intuitive UI, lightning-fast workflow.

Powerful Code View

Advanced users can drop into the browser-based code editor with syntax highlighting, autocompletion, and full DSL control.

Room Setup Wizard

Start your project with a guided wizard: define floors, rooms, zone types — and OpenLogic builds the initial module structure for you.

Community Module Library

Install logic modules, device drivers, and UI components from other integrators. Or share your own with the OpenLogic community.

Modern UI Designer

Design and deploy clean, responsive touchscreen interfaces. Start with built-in themes or build your own layouts visually.

Bridge Everything

Support for Home Assistant, Crestron Home, Lutron, Sonos, Spotify, Jellyfin, and more — with a bridge architecture that's easy to extend.

Portable Runtime

Run OpenLogic on Docker, Linux, Raspberry Pi, or embedded controller hardware. Minimal footprint, maximum flexibility.

Built in Public

Follow our roadmap, contribute modules, and join the future of integrator-built home control.
