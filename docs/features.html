Features for Professional Integrators

  • 🔧 Modular Architecture: Create reusable, structured logic modules with simple, declarative syntax.
  • 🧠 Clean Signal Logic: Inspired by Crestron’s SIMPL+ model — refined and modernized.
  • 🛠️ Built-In Logic Library: Complete collection of symbols (AND, NOT, Toggle, Interlock, etc.) ready to go.
  • 🖥️ Custom UI Designer: A browser-based visual UI tool is in the works for quick layout and control workflows.
  • 🌉 System Integration: Interface with Crestron Home, Home Assistant, Lutron, and more through bridge modules.
  • ⚙️ Open Runtime: Works in Docker, Linux, NUC, or Pi. Build locally, deploy anywhere.
  • 🚀 Built in Public: Developed openly on GitHub with a growing community of integrators and developers.
